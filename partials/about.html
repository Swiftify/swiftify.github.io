Swiftify

Generate Swift model classes from example JSON data.

Usage

  1. Go to JSON tab, and paste valid JSON object. You can use provided examples to see how it works.
  2. Go to Generated Code and copy-paste classes to your project.

With Swiftify you can get any JSON response from web service you want to interact with, and generate Swift model classes. This tool has been inspired by Objectify app, whose development has, sadly, stopped.

Swiftify is single-page web application written in JavaScript using AngularJS toolkit. It runs here, in your browser.

Features

  • Uses SwiftyJSON for reading JSON as safe bet. Other parsers may be supported in future.
  • Generates classes for nested objects.
  • Infer numeric data type from example, Double or Int, depending on number has fractional part, or not.
  • Generates type-safe Swift arrays of nested objects or primitive types.
  • Objects are deserialized in init(json: JSON) initializer or custom function named by user.
  • Generated field and class names can be changed to your convenience.
