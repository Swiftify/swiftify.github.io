Swiftify

Generate Swift model classes from example JSON data.

With Swiftify you can pick any JSON payload from web service you want to interact with, and generate Swift model classes. This tool has been inspired by Objectify app, whose development has stopped)

Initialize objects with SwiftyJSON's JSON object.

Features

  • Generates classes for nested objects.
  • Infer numeric data type from example, Double or Int, depending on number has fractional part, or not.
  • Generates type-safe Swift arrays of nested objects or primitive types.
  • Uses SwiftyJSON for reading JSON. Other parsers may be supported in future.

Usage

  1. Go to JSON tab, and paste valid JSON object.
  2. Go to Generated Code and copy-paste classes to your project.
